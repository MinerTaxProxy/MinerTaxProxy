本软件支持所有币种与矿池的中转服务，软件永久免费！
使用方法:将本软件放置与windos系统的服务器内运行，矿池地址填写需要转发的矿池地址，矿池地址只需要填入矿池的域名，不要加端口，服务器地址需填写本机服务器内网IP，端口写在后面矿池端口去，一般服务器端口与矿池端口填写一致或服务器端口自行设置都可以，但矿池端口必须是矿池提供的端口，例如鱼池给予的6688端口，设置完毕后，点击启动即可！傻瓜一键式操作!
抽水功能：在启动服务之前需要勾选抽水比例，抽水比例的单位为‰，所以填写抽水比例时，如百分之1抽成，则需要填写10‰，钱包地址填写个人的抽水地址即可！
